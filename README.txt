4、使用正则表达式,查找需要添加文字链接、表情的字符串 
5、我们使用绘图的方式绘制表情面板
6、放大缩小微博列表中的缩略图,并且实现大图的网络加载,缓存处理,GIF 图片的 动画显示 7、使用单例方法获取对象,应用对象方法将主题相关信息存储到沙盒中并提取
